# 09 开发日志

## v0.1.0-phase1 - 2026-07-08

阶段：基础工程与数据库

### 完成内容

- 创建 FastAPI 后端项目结构。
- 增加 SQLAlchemy 数据库连接。
- 增加环境变量配置 `.env.example`。
- 增加 Dockerfile 和 `docker-compose.yml`。
- PostgreSQL 宿主机端口使用 `5433`，避免和本机已有 PostgreSQL 的 `5432` 冲突。
- Redis 宿主机端口使用 `6380`，避免和本机已有 Redis 的 `6379` 冲突。
- 增加 PostgreSQL 初始化 SQL。
- 创建需求文档中第一阶段要求的 21 张表。
- 初始化 ETF 资产池、默认策略配置、基础风控规则。
- 增加 `GET /health` 健康检查接口。
- 增加 `GET /api/assets` ETF 主数据查询接口。
- 增加项目文档体系。
