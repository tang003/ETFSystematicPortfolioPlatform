# 05 策略设计

当前阶段尚未实现策略引擎，只完成 `strategy_config` 表和默认策略配置种子数据。

## 默认策略

策略代码：`core_etf_rotation`  
调仓频率：`monthly`

默认因子权重：

```json
{
  "trend": 0.35,
  "momentum": 0.30,
  "volatility": 0.15,
  "drawdown": 0.10,
  "liquidity": 0.05,
  "premium": 0.05
}
```

## 后续实现要求

策略参数必须从 `strategy_config.config` 读取，不允许写死在策略代码中。

策略运行必须创建 `strategy_run`，所有信号、目标仓位、风控结果、调仓单和报告都需要通过 `run_id` 追溯。

综合评分公式：

```text
alpha_score =
trend_score * 0.35
+ momentum_score * 0.30
+ volatility_score * 0.15
+ drawdown_score * 0.10
+ liquidity_score * 0.05
+ premium_score * 0.05
```
