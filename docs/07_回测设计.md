# 07 回测设计

当前已实现基础等权买入持有回测，用作策略开发前的基线验证。

## 相关表

- `backtest_run`：回测任务。
- `backtest_equity_curve`：净值曲线。
- `backtest_trade`：交易记录。
- `backtest_metrics`：绩效指标。

## 后续默认参数

```text
initial_cash = 10000
monthly_contribution = 2000
fee_rate = 0.001
slippage_rate = 0.001
rebalance_frequency = monthly
```

## 后续指标

至少输出：

- 累计收益率
- 年化收益率
- 最大回撤
- 波动率
- 夏普比率
- 卡玛比率
- 胜率
- 月度胜率
- 换手率
- 交易次数

## 已实现回测模式

模式：`equal_weight_buy_and_hold`

流程：

```text
读取 market_data_clean
→ 选择 symbols
→ 首日等权买入
→ 按每日 close 计算组合市值
→ 记录 backtest_equity_curve
→ 记录初始交易 backtest_trade
→ 计算 backtest_metrics
```

当前支持参数：

- `symbols`
- `start_date`
- `end_date`
- `initial_cash`
- `monthly_contribution`
- `fee_rate`
- `slippage_rate`

当前已实现指标：

- `cumulative_return`
- `annualized_return`
- `max_drawdown`
- `volatility`
- `sharpe_ratio`
- `trade_count`
