# 06 风控设计

当前阶段尚未实现风控引擎，只完成 `risk_rule` 和 `risk_check_result` 表结构，以及基础风控规则种子数据。

## 已初始化规则

- 单一 ETF 最大仓位：`max_single_weight <= 50%`
- 权益资产最大仓位：`max_equity_weight <= 80%`
- 防御资产最低仓位：`min_defense_weight >= 20%`
- 现金最低仓位：`min_cash_weight >= 5%`
- 跨境 ETF 新增买入溢价限制：`premium_buy_limit <= 3%`

## 后续实现要求

风控引擎必须独立于策略引擎：

```text
策略生成 raw_target_weight
→ 风控读取 risk_rule
→ 保存 risk_check_result
→ 输出 final_target_weight
```

所有风控修正必须保存原因，不能只覆盖结果。
